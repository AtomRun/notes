# SpringBoot概述

## SpringBoot是什么，能做什么？

- 是一个快速开发的脚手架
- 作用：快速创建独立的、生产级的基于Spring的应用程序

## 特性

- 无需WAR文件
- 提供starter简化配置
- 尽可能自动配置Spring以及第三方库
- 无代码生成&无XML
- 提供“生产就绪”功能，例如指标、健康检查、外部配置等
